# Data

The ICLE corpus (Granger et al., 2020) is available under license from the Centre for English Corpus Linguistics, UCLouvain:
https://uclouvain.be/en/research-institutes/ilc/cecl/icle.html

The LOCNESS corpus is available from the same source.

The `results/` directory contains all intermediate and final outputs from the LC Framework pipeline, enabling verification of all reported findings without access to the raw corpus data.
